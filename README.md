
# Software Asset Validation and Lifecycle Assessment

## Overview

This project analyzes software asset records using Python and Pandas.

It helps identify expired software licenses, validate records, detect duplicates, and generate automated reports.

---

## Features

* Detect missing values
* Detect duplicate records
* Identify expired software
* Generate automated CSV reports
* Department-wise analytics visualization

---

## Technologies Used

* Python
* Pandas
* Matplotlib

---

## Project Workflow

1. Load dataset from CSV
2. Validate records
3. Analyze expired software
4. Generate reports
5. Visualize department-wise insights

---

## Output

* Expired software report
* Bar chart visualization

---

## Files Included

* `main.py`
* `software_assets_dataset.csv`
* `expired_software_report.csv`
* `requirements.txt`
